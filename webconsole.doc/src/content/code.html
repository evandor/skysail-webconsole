title=Code date=2015-10-29 type=page status=published ~~~~~~ This section covers details about skysails codebase.

If you're a user of skysail application, you most propably will not find this very interesting; as a future developer of such applications, you should have a look ;)

The Application Model describes the core domain of the skysail server.

An application is a collection of entities, (aggregate) repositories and references between them. Entities themselves consist of fields, which have types. The purpose of this structure is to provide a common model for setting up executable APIs.

Application

An application has a (unique) name, a collection of entities and a collection of repositories.

Entity

An Entity describes a "Thing" of the real world or any concept.
